### 常用图表类型

| 图表类型 | 用途 | 复杂度 |
|---------|------|--------|
| 流程图 | 业务流程、算法流程 | ⭐ |
| 时序图 | 系统交互、API 调用 | ⭐⭐ |
| 甘特图 | 项目计划、时间管理 | ⭐⭐ |

| 类图 | 面向对象设计、架构 | ⭐⭐⭐ |

## 💡 使用建议

1. **选择合适的图表类型** - 根据要表达的内容选择最合适的图表
2. **保持简洁** - 避免过于复杂的图表，影响可读性
3. **添加说明** - 在图表下方添加文字说明，帮助理解
4. **版本控制** - 图表代码存储在 Git 中，便于版本管理
