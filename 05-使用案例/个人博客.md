# 个人技术博客案例

本文介绍如何使用 PowerWiki 构建个人技术博客。

## 博客定位

- **目标读者**: 开发者、技术爱好者
- **内容方向**: 技术教程、学习笔记、项目经验
- **更新频率**: 每周 1-2 篇

## 目录结构

```
my-blog/
├── README.md                  # 博客首页
├── ABOUT.md                   # 关于我
├── images/                    # 全局图片
│
├── 前端开发/
│   ├── README.md
│   ├── images/
│   ├── React学习笔记.md
│   ├── Vue3新特性.md
│   └── CSS技巧汇总.md
│
├── 后端开发/
│   ├── README.md
│   ├── images/
│   ├── Node.js实战.md
│   ├── Python入门.md
│   └── 数据库设计.md
│
├── 运维部署/
│   ├── README.md
│   ├── Docker从入门到实践.md
│   ├── Nginx配置详解.md
│   └── CI-CD部署流程.md
│
└── 项目经验/
    ├── README.md
    ├── 博客系统开发.md
    └── 电商项目总结.md
```

## 配置文件

```json
{
  "gitRepo": "https://github.com/yourusername/my-blog.git",
  "repoBranch": "main",
  "port": 3150,
  "siteTitle": "我的技术博客",
  "siteDescription": "记录学习、分享技术",
  "language": "zh-CN",
  "autoSyncInterval": 180000,
  "pages": {
    "home": "README.md",
    "about": "ABOUT.md"
  }
}
```

## 内容策略

### 1. 文章类型

| 类型 | 占比 | 示例 |
|------|------|------|
| 教程 | 40% | 从零开始学习 React |
| 笔记 | 30% | JavaScript 技巧总结 |
| 项目 | 20% | 博客系统开发记录 |
| 工具 | 10% | 效率工具推荐 |

### 2. SEO 优化

每篇文章添加完整的 Frontmatter：

```yaml
---
title: React Hooks 完全指南
description: 深入理解 React Hooks 的使用方法和最佳实践
author: 你的名字
date: 2026-02-07
keywords: React, Hooks, 前端开发
tags: [React, 前端, 教程]
---
```

### 3. 图片管理

使用相对路径引用图片：

```markdown
![React Hooks](./images/react-hooks.png)
```

## 部署方式

### Docker 部署（推荐）

```bash
docker run -d -p 3150:3150 \
  -v /path/to/config.json:/app/config.json \
  sayunchuan/powerwiki
```

### 免费托管

将仓库连接到 Vercel 或 Netlify 实现免费托管。
