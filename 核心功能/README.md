# PowerWiki 核心功能

PowerWiki 提供了丰富的功能，让您轻松构建和管理知识库。

## 📚 功能总览

### 1. Markdown 支持
完整的 Markdown 语法支持，包括基础语法和扩展功能。

**查看**: [Markdown 语法演示](../功能演示/Markdown语法演示.md)

**特性**:
- 标题、列表、表格
- 代码块、引用、链接
- 脚注、任务列表
- HTML 嵌入

### 2. 代码高亮
支持 100+ 种编程语言的语法高亮。

**查看**: [代码高亮演示](../功能演示/代码高亮演示.md)

**支持语言**:
- JavaScript、Python、Java、Go、C++
- SQL、HTML/CSS、Bash、Shell
- JSON、YAML、TypeScript、Markdown

### 3. Mermaid 图表
强大的图表支持，无需外部工具。

**查看**: [图表演示](../功能演示/图表演示/)

**图表类型**:
- 流程图 (Flowchart)
- 时序图 (Sequence Diagram)
- 甘特图 (Gantt Chart)
- 类图 (Class Diagram)
- 状态图 (State Diagram)
- 实体关系图 (ER Diagram)

### 4. Git 集成
自动从 Git 仓库同步内容，天然支持版本控制。

**特性**:
- 自动同步
- 版本历史
- 多仓库支持
- 分支管理

### 5. 图片管理
灵活的图片引用和管理方式。

**特性**:
- 本地图片支持
- 相对路径引用
- 自动路径转换
- 多种图片格式

### 6. SEO 优化
完整的 SEO 优化支持。

**查看**: [SEO 优化演示](../功能演示/SEO优化演示.md)

**特性**:
- Frontmatter 元数据
- 结构化数据
- Sitemap 生成
- Open Graph 支持

### 7. 响应式设计
完美适配所有设备。

**查看**: [响应式设计演示](../功能演示/响应式设计演示.md)

**特性**:
- 移动优先设计
- 多设备适配
- 触摸友好
- 快速加载

### 8. 浏览统计
自动追踪文章浏览次数。

**特性**:
- 浏览计数
- 访问统计
- 热门文章

### 9. PDF 支持
支持 PDF 文件渲染。

**特性**:
- PDF 转图片
- 在线预览
- 下载支持

### 10. 多语言支持
支持多种语言界面。

**支持语言**:
- 中文 (zh-CN)
- 英文 (en)
- 可扩展

## 💡 使用建议

### 对于个人用户
1. 使用 Markdown 编写文档
2. 利用 Git 进行版本控制
3. 使用 Mermaid 绘制图表
4. 优化 SEO 提高可见性

2. 使用 Git 工作流协作

2. 配置 SSL 证书
3. 设置备份策略
4. 监控和日志管理
