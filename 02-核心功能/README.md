# PowerWiki 核心功能

PowerWiki 提供了丰富的功能，让您轻松构建和管理知识库。

## 功能总览

### 1. Markdown 支持
完整的 Markdown 语法支持，包括基础语法和扩展功能。

**查看演示**: [Markdown 语法演示](../01-功能演示/Markdown语法演示.md)

### 2. 代码高亮
支持 100+ 种编程语言的语法高亮。

**查看演示**: [代码高亮演示](../01-功能演示/代码高亮演示.md)

### 3. Mermaid 图表
强大的图表支持，无需外部工具。

**查看演示**: [图表演示](../01-功能演示/图表演示/)

### 4. Git 集成
自动从 Git 仓库同步内容，天然支持版本控制。

**特性**:
- 自动同步
- 版本历史
- 多仓库支持
- 分支管理

### 5. 图片管理
灵活的图片引用和管理方式。

**特性**:
- 本地图片支持
- 相对路径引用
- 自动路径转换
- 多种图片格式

### 6. SEO 优化
完整的 SEO 优化支持。

**查看演示**: [SEO 优化演示](../01-功能演示/SEO优化演示.md)

### 7. 响应式设计
完美适配所有设备。

**查看演示**: [响应式设计演示](../01-功能演示/响应式设计演示.md)

### 8. 浏览统计
自动追踪文章浏览次数。

### 9. PDF 支持
支持 PDF 文件渲染。

### 10. 多语言支持
支持 8 种语言界面：中文、英文、日文、韩文、西班牙文、法文、德文、俄文。
